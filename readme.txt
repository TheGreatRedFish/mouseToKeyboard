Description: 
	Program for moving mouse with keyboard. 
Usage:
	Hot Keys: 
		` - on\off
		q - off 
		w - up 
		d - down 
		a - left 
		d - right
		e - right click
		f - left click
	Install: 
		1) clone or just download repo
		2) $python -m venv venv
		3) win ($venv\Scripts\activate) | linux ($source venv/bin/activate)
		4) pip install -r requirements.txt
		5) python main.py
